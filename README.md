# Unity-Agent

Unity-Agent is an agentic development framework for Unity projects.

## Current capabilities

- Reads Unity project files
- Diagnoses bugs
- Generates patch proposals
- Sends proposals to Discord
- Waits for human approval
- Applies approved patches with Git backups

## Current supported project

- AinaQuest

## Setup

1. Clone the repo
2. Create a virtual environment
3. Install dependencies
4. Copy `.env.example` to `.env`
5. Fill in required API keys
6. Run the main orchestrator

## Example environment variables

See `.env.example`
